# knowhere

Personal Claude Code plugin marketplace for bruk-io projects.

## What is knowhere?

knowhere is a plugin marketplace for [Claude Code](https://claude.com/claude-code) — it provides a curated collection of plugins that scaffold projects, generate CI/CD workflows, and enforce development patterns via copier templates.

## Available Plugins

| Plugin | Category | Description |
|--------|----------|-------------|
| **claude-code** | development | CLAUDE.md configuration scaffolding |
| **containers** | deployment | Dockerfile generation for Python, Node.js, Go, Rust |
| **github** | deployment | CI and GitHub Pages workflow generation |
| **go** | development | Go project scaffolding with Makefile |
| **knowhere-publisher** | development | Set up repos to publish to this marketplace |
| **mkdocs** | development | MkDocs Material documentation scaffolding |
| **node** | development | Node.js/TypeScript project scaffolding |
| **obsidian-plugin-dev** | development | Obsidian plugin development scaffolding |
| **python** | development | Python project scaffolding (UV, Ruff, Hatchling) |
| **rust** | development | Rust project scaffolding with Makefile |
| **textual** | development | Textual TUI framework scaffolding |
